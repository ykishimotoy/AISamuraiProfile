■ サイトの目的
━━━━━━━━━━━━━━━━━━━━━━
本サイトは、生成AIを軸にした教育・実務・コミュニティ活動を行う個人の

- 日本語を主とし、英語版も併記する

  企業・行政・教育関係者も想定する
- 技術デモではなく「文章と思想」が主役のサイトとする

━━━━━━━━━━━━━━━━━━━━━━
■ 全体要件（重要）
━━━━━━━━━━━━━━━━━━━━━━
1. **静的HTMLで完結すること**
   - フレームワーク不要（React / Vue 等は使わない）
   - GitHub Pages でそのままホスト可能な構成

2. **JA / EN 切り替え機能を実装すること**
   - ページ遷移なしで、ボタン操作により言語が切り替わる
   - 日本語・英語はそれぞれ `<div>` 単位で全文を保持する
   - `display: none` / `display: block` 等で表示切替を行う
   - 選択した言語は `localStorage` に保存し、再訪時に保持されること
   - 初期表示は日本語（JA）

3.　**デザイン**
   - デザインは任せる。「AIが作ったんでしょー」と感じさせるサイトデザインはダメ。デザイン面をめっちゃバチくそに頑張って素晴らしくカッコよくしてくれ。20年選手のデザイナーとしてガチガチに考え抜いて、これ以上ないってデザインに刷新して。俺は君に超絶期待してる、めちゃくちゃ頑張ってくれ。
